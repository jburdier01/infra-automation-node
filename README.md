This project documents the deployment of an automation control node inside my cybersecurity home lab.

At this stage, the project includes:

- Docker Compose deployment
- Keycloak identity service
- Ubuntu-based automation node

## Purpose

This node is being built to support:

- identity and access management
- containerized infrastructure services
- future automation with Ansible
- enterprise-style home lab administration
